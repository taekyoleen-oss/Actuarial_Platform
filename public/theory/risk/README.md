# 보험이론 사전 — 위험·파산이론 자료 폴더

이 폴더에 `.html` / `.pdf` 파일을 추가하고 커밋·푸시하면, 재배포 시
사이트의 **보험이론 사전 → 위험·파산이론** 목록에 자동으로 표시됩니다.

## 규칙

- 목록에는 **`.html`이 있는 자료만** 표시됩니다(클릭 시 본문 열람).
  같은 파일명의 `.pdf`는 함께 보관할 수 있지만 **현재 사이트에는 노출하지 않습니다**.
- 같은 파일명의 `.svg`(커버 일러스트)는 게시 요청 시 **theory-publisher 스킬이 자동 생성**합니다 — 직접 만들 필요 없습니다.
- 목록 제목은 HTML `<title>`의 `한글명 (영문명)` 부분을 우선 사용합니다
  (`한글명 (영문명) — 학습 해설서` 형식 권장). `<title>`이 없으면 파일명에서
  자동 정리됩니다: 숫자 접두사(`01_`) 제거 · `_` → 띄어쓰기 · "해설서" 단어 제거.
- 정렬은 파일명 가나다순입니다(숫자 접두사로 순서 지정 가능).
- HTML이 참조하는 이미지 등 부속 파일도 이 폴더에 함께 넣으면 됩니다.
- 파일명에 `%` 문자는 피해주세요.
