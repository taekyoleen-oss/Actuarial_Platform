# 보험이론 사전 — 손해보험 자료 폴더

이 폴더에 `.html` / `.pdf` 파일을 추가하고 커밋·푸시하면, 재배포 시
사이트의 **보험이론 사전 → 손해보험** 목록에 자동으로 표시됩니다.

## 규칙

- **파일명(확장자 제외)이 같은** `자료명.html` + `자료명.pdf` 는 **한 항목**으로 묶입니다.
  HTML은 사이트 안에서 바로 열람, PDF는 내려받기 링크로 함께 노출됩니다.
- 목록의 **제목 = 파일명**(확장자 제외). 앞에 `01_` 처럼 숫자 접두사를 붙이면
  **정렬 순서로만 쓰이고 제목에서는 숨겨집니다**. (예: `01_손해보험의 기초.html` → "손해보험의 기초")
- 정렬은 파일명 가나다순입니다.
- HTML이 참조하는 이미지 등 부속 파일도 이 폴더에 함께 넣으면 됩니다(목록에는 html/pdf만 표시).
- 파일명에 `%` 문자는 피해주세요.
